## rpg_classLesson
- 言語：Python
- 概要：RPGにおいて武器を装備するとキャラのステータスが変動するところまでを再現
- 備考：
class の概念を手に馴染ませるために練習  
インスタンスを増やしていくうちに、「csvから一気に読み込む方が実用的なのでは」と気づき、ChatGPTを用いてそれに必要な考え方やモジュール、関数などを調べて実装した

ゆくゆくは防具も同様に装備して、敵モンスターを実装し、敵とこちらとのパラメータの相性によって勝敗がつくようなゲームに発展させたい

## game_lesson
言語：HTML/JavaScript
概要：自キャラと障害物があり、自キャラをジャンプさせることで障害物を回避します。
備考：classを利用して変数をなるべく減らし、可読性を意識した。
     接触判定を簡単にするために、各キャラを点とみなし、三平方の定理を利用した。
     シーンを設定し、メイン画面とゲームオーバー画面を実装した。
参考：https://www.youtube.com/watch?v=-y5OJA9z8bU
